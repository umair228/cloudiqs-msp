## Troubleshooting

### "Module not found" errors?
```bash
# Clean install
rm -rf node_modules package-lock.json
npm install
```

### Build fails with CI errors?
```bash
# Use CI=false flag
CI=false npm run build
```

### Port 3000 already in use?
```bash
# Kill the process or use different port
PORT=3001 npm start
```
